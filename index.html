📋 Generador Cuadrante Tareas Profesorado

Asigna profesores a tareas de apoyo con optimización CP-SAT — ideal para procesos como XarxaLlibres

🌟 Ver en GitHub 🚀 Instalar

✨ ¿Qué hace?

Esta aplicación de escritorio genera cuadrantes de apoyo al profesorado respetando restricciones reales de los centros educativos: disponibilidad horaria, límites de carga, solapamientos y mínimos/máximos de profesores por tarea.

📖 Flujo de trabajo

1

👨‍🏫 Profes — Añade profesores con su disponibilidad horaria y límites de carga

2

🏠 Proyecto — Crea o selecciona un proyecto (duplica existentes para variantes)

3

📋 Necesidades — Añade tareas con fecha, hora y mínimo/máximo de profesores

4

⚙️ Generar — El solver CP-SAT genera 5 opciones y tú eliges la mejor

5

📅 Cuadrante — Abre el HTML en el navegador, copia los correos, imprime como PDF

📦 Instalación

Clona el repositorio e instala las dependencias:

git clone https://github.com/tu-usuario/GeneradorCuadranteTareasProfesorado.git
cd GeneradorCuadranteTareasProfesorado
pip install -r requirements.txt
python main.py

O simplemente haz doble clic en lanzar.bat (Windows).

🧠 Solver CP-SAT

El modelo de optimización combinatoria con Google OR-Tools:

Variables:  x[necesidad, profesor] ∈ {0, 1}

Restricciones:
  • Cada necesidad recibe entre min y max profesores
  • Profesor disponible en la franja horaria
  • Sin solapamientos para un mismo profesor
  • Horas totales ≤ límite del profesor
  • Horas por día ≤ límite diario

Objetivo:  Maximizar Σ x (más asignaciones posible)
