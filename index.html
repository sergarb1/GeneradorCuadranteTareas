✨ ¿Qué hace?

Aplicación de escritorio que genera cuadrantes de asignación de profesorado a tareas respetando disponibilidad horaria, límites de carga, solapamientos y mínimos/máximos de profesores por tarea. Usa Google OR-Tools CP-SAT para optimización combinatoria.

🧠 Solver CP-SAT

El modelo de optimización combinatoria con Google OR-Tools:

Variables:  x[necesidad, profesor] ∈ {0, 1}

Restricciones:
  • Cada necesidad recibe entre min y max profesores
  • Profesor disponible en la franja horaria
  • Sin solapamientos para un mismo profesor
  • Horas totales ≤ límite del profesor
  • Horas por día ≤ límite diario

Objetivo:  Maximizar Σ x (más asignaciones posible)

🛠️ Tecnologías

  • Python 3.11+

  • Color HEX personalizado con selector visual
  • Email opcional (se usa como ATTENDEE en ICS)
  • Límite de horas semanales y por día
  • Franjas con botones predefinidos (Mañana, Tarde, Completo)
  • Franjas personalizadas y plantillas de horarios
  • Duplicar profesor, duplicar franja, duplicar a otro día
  • Búsqueda y filtro, importar/exportar JSON

  • Importar/exportar JSON

🧠 Solver

  • Loading animado con spinner
