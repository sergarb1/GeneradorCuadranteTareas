🌟 Ver en GitHub 🚀 Instalar 📖 Manual de usuario

✨ ¿Qué hace?

Esta aplicación de escritorio genera cuadrantes de apoyo al profesorado respetando restricciones reales de los centros educativos: disponibilidad horaria, límites de carga, solapamientos y mínimos/máximos de profesores por tarea. Usa Google OR-Tools CP-SAT para optimización combinatoria.

📖 Flujo de trabajo

1

👨‍🏫 Profes — Añade profesores con disponibilidad, turno y color (o importa JSON)

2

🏠 Proyecto — Crea o selecciona un proyecto (o importa proyecto completo JSON)

3

📋 Necesidades — Añade tareas con fecha, hora y mínimo/máximo (o importa JSON)

4

⚙️ Generar — Configura de 1 a 50 opciones, el solver calcula con validación previa

5

🔒 Cuadrante — Bloquea profes a tareas (clic), regenera, exporta HTML/PNG/CSV

6

📅 Navega — Cambia entre opciones con ◀ ▶, toolbar agrupada (Generar HTML · Acciones · Exportar)

📦 Instalación

Clona el repositorio e instala las dependencias:

git clone https://github.com/tu-usuario/GeneradorCuadranteTareasProfesorado.git
cd GeneradorCuadranteTareasProfesorado
pip install -r requirements.txt
python main.py

O simplemente haz doble clic en lanzar.bat (Windows).

🧠 Solver CP-SAT

El modelo de optimización combinatoria con Google OR-Tools:

Variables:  x[necesidad, profesor] ∈ {0, 1}

Restricciones:
  • Cada necesidad recibe entre min y max profesores
  • Profesor disponible en la franja horaria
  • Sin solapamientos para un mismo profesor
  • Horas totales ≤ límite del profesor
  • Horas por día ≤ límite diario

Objetivo:  Maximizar Σ x (más asignaciones posible)

🛠️ Tecnologías

  • Python 3.11+ — Lenguaje principal
  • PyQt6 — Interfaz gráfica moderna con Fusion style y QSS
  • OR-Tools CP-SAT — Optimización combinatoria de Google
  • HTML + CSS — Exportación responsiva optimizada para impresión

📋 Funcionalidades detalladas

👨‍🏫 Gestión de profesores

  • Los profesores son globales (compartidos entre todos los proyectos)
  • Asignación de turno preferente: Cualquiera / Mañana / Tarde
  • Color HEX personalizado con selector visual
  • Límite de horas semanales y por día
  • Franjas de disponibilidad con botones predefinidos (Mañana 09-14h, Tarde 15-18h, Completo 09-18h)
  • Franjas personalizadas con diálogo de hora libre
  • 📋 Duplicar profesor: copia exacta con todas sus franjas, turno y color
  • 📋 Duplicar franja: copia una franja para ajustar su fecha u hora
  • 📋 Duplicar franjas a otro día: copia en bloque todas las franjas del profesor a otra fecha
  • Plantillas de horarios: guardar y cargar combinaciones de franjas
  • Búsqueda y filtro por nombre
  • Importar / exportar profesores en JSON

📋 Necesidades (tareas)

  • Se definen por proyecto con fecha, hora de inicio y fin
  • Mínimo y máximo de profesores simultáneos por tarea
  • 📋 Duplicar necesidad: copia exacta para crear variantes rápidamente
  • 📋 Duplicar necesidades a otro día: copia en bloque todas las necesidades de una fecha a otra
  • Búsqueda y filtro por nombre o fecha
  • Importar / exportar necesidades en JSON

🧠 Solver CP-SAT

  • Opciones configurables (1-50) generadas con distintas semillas de búsqueda
  • Loading animado con spinner mientras el solver resuelve (evita dobles clics)
  • Validación previa: avisa si hay necesidades sin cobertura u horas insuficientes
  • Bloqueo de asignaciones: clic en un profesor en el cuadrante para fijarlo a esa tarea
  • Asignación manual: clic derecho en una tarea para añadir o quitar profesores sin regenerar
  • Regeneración respetando bloqueos
  • Máximo 10 segundos por opción (solución factible garantizada)

📅 Cuadrante

  • Vista calendario por día con tarjetas de tarea y profesores coloreados
  • Vista normal / compacta (toggle "Cambiar a vista...")
  • Navegación entre opciones con ◀ ▶
  • Bloquear / desbloquear asignaciones con un clic
  • Asignación manual: clic derecho para añadir o quitar profesores sin regenerar
  • Arrastrar profesor: suelta el nombre de un profe en otra tarea para reasignarlo
  • Editar necesidad: menú contextual → "📝 Editar necesidad" abre el diálogo de edición
  • Duplicar día: botón "📋 Duplicar día" copia necesidades entre fechas para editar y regenerar
  • Lock highlight: asignaciones bloqueadas con fondo 0.35 opacity + borde 2px del color del profe
  • Estadísticas: carga por profesor, cobertura global, horas totales
  • Toolbar en 3 grupos: Generar HTML (botón + combo vista), Acciones (Regenerar Bloqueos, Carpeta, Stats, Compacta), Exportar (DOCX, Calendario ICS, PNG)
  • Scrollbars finos (6px) para máximo espacio útil

📦 Proyectos

  • Persistencia local en projects/*.json
  • Crear, duplicar, guardar y eliminar proyectos
  • Las opciones generadas se guardan automáticamente al proyecto
  • Al recargar un proyecto, se restauran las 10 opciones
  • Importar / exportar proyecto completo (profesores + necesidades)
  • Exportar CSV con todas las asignaciones

📤 Exportación

  • HTML con 5 vistas seleccionables: 📅 Cuadrante, 👤 Por profesor, 📋 Por tarea, 📄 Estilo Word (copia/pega a Word), 🖨️ Para imprimir
  • Correos electrónicos generales e individuales integrados en el HTML con botón "📋 Copiar"
  • DOCX: documento Word con solo los cuadrantes (tablas limpias, sin extras)
  • ICS: calendario para Google Calendar / Outlook
  • PNG: captura directa del cuadrante
  • CSV: datos tabulares para hoja de cálculo
  • JSON: profesores, necesidades o proyecto completo

🎨 Interfaz

  • Tema claro / oscuro / alto contraste (♿) — ciclo de 3 modos
  • Deshacer / Rehacer con botones con texto (hasta 50 estados)
  • Auto-guardado cada 2 minutos
  • Notificaciones toast no intrusivas
  • Panel de log con información detallada del solver
  • Layout adaptado a 1280×720, columnas estrechas (190px) para que quepan 5 días
  • Scrollbars finos (6px) para máximo espacio útil
  • Tooltips descriptivos en todos los botones e inputs
  • Barra de estado inferior con mensajes contextuales y atajos
  • Atajos de teclado: Ctrl+S, Ctrl+Z, Ctrl+Y, Ctrl+N, Ctrl+D, F1, Esc
  • Diálogo de bienvenida interactivo para nuevos usuarios
  • Botón 📖 Ayuda en la cabecera que abre el manual completo
